# WEZA Build Brand Kit

## Brand Idea

WEZA Build is a construction payout operating system: grounded, exact, and auditable. The brand should feel like site documentation meeting financial infrastructure. No crypto gloss, no generic fintech purple, no construction cliche.

## Logo

The mark is a compact "W" drawn like a milestone graph and structural truss. The emerald stroke represents approved workflow; the amber baseline represents release of payment; the dark tile represents the audit ledger.

Assets:

- `public/brand/weza-mark.svg` - square app icon / favicon / avatar.
- `public/brand/weza-wordmark.svg` - horizontal wordmark.
- `public/brand/weza-logo-card.svg` - 16:9 submission graphic for hackathon directories and social previews.

## Colors

- Ledger Black: `#070910` - primary background, audit trail, logo tile.
- Site Ink: `#0E121C` - primary text.
- Approval Green: `#34D399` - approval, success, Solana proof.
- Payout Amber: `#F59E0B` - payout release and action accents.
- Paper White: `#F7F8FA` - page surfaces.
- Steel Gray: `#3E4A5E` - secondary text.

## Typography

Use strong grotesk or industrial sans-serif typography for headings. In the app, the current stack stays practical for performance, but pitch graphics should use:

- Primary: Inter, Avenir Next, or Manrope.
- Mono: JetBrains Mono for transaction signatures, IDs, and audit proof.

## Voice

Short, direct, operational.

Use:

- "Approval-to-payout for construction."
- "Off-chain workflow. On-chain payout proof."
- "The payout moment is the on-chain moment."
- "Drawings stay private. Payment proof stays public."

Avoid:

- "Everything is on-chain."
- "Escrow" or "custody."
- "AI-powered."
- "Mainnet settlement."
